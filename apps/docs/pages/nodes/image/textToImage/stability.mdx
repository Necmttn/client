## Path
`flow.image.textToImage.stability`

## Description

Takes a string (`text`) and a `modifier` and replaces the string `$(text)$` in the `modifier` with the `text` field.

## API Key

Uses the `stability` API key:

```ts index.ts
const aigur = createClient({
  apiKeys: {
    stability: process.env.STABILITY_API_KEY
  }
})
```

## Example

```ts index.ts
//...
flow.text.modify.simple(() => ({
  text_prompts: [
    {
      text: 'a green apple'
    }
  ]
})) // --> {result: ArrayBuffer}
```
