In order to reduce bundle size, installation time and clutter some nodes will not be in the core `@aigur/client` package. Instead, they will be in separate packages and will be used as [custom nodes](/customnodes).

- `@aigur/supabase` - Nodes that interact with [Supabase](https://supabase.com) database.

```ts index.ts
import { supabaseUpload } from '@aigur/supabase';

// ...
flow.custom(supabaseUpload)(({ prev }) => ({
  bucket: 'results',
  extension: 'png',
  file: prev.result,
  supabaseServiceKey: process.env.SUPABASE_SERVICE_KEY!,
  supabaseUrl: process.env.SUPABASE_URL!,
}))
// ...
```
