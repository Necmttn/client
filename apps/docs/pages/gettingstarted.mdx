# Installation

<Tabs items={['pnpm', 'npm', 'yarn']}>
	<Tab>
		```fish copy
		pnpm add @aigur/client
		```
	</Tab>
	<Tab>
		```fish copy
		npm i @aigur/client
		```
	</Tab>
	<Tab>
		```fish copy
		yarn add @aigur/client
		```
	</Tab>
</Tabs>

Import the client in your project:
```ts copy
import { createClient } from '@aigur/client';
```

Create a new client:
```ts copy
const aigur = createClient({
	apiKeys: {
		// Your API keys
	}
});
```



<Callout>It is recommended to create Aigur's client on the server in order not to leak your API keys in the client (browser).</Callout>
